这是我在 Vingues 写下的第一篇文章。这个博客使用 [Astro](https://astro.build) 构建——一个专注于内容驱动网站的现代框架。

## 为什么选择 Astro

在尝试了众多框架之后，Astro 的理念深深吸引了我：

- **零 JS 默认** — 页面不加载不必要的 JavaScript
- **Islands 架构** — 只在需要交互的地方注入运行时
- **内容优先** — Markdown / MDX 是一等公民
- **极致性能** — 构建结果是纯静态 HTML

## 代码示例

一个简单的 Astro 组件：

```astro
---
const greeting = "Hello, Astro!";
---

<h1>{greeting}</h1>
```
